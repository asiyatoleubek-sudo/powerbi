# Power BI Dashboard Portfolio

## Overview
This repository contains multiple interactive dashboards created in Power BI using datasets from Kaggle. The projects focus on data visualization, analysis, and insight generation across different domains such as music, fitness, coffee, and streaming platforms.

## Dashboards
- Coffee Data Analysis Dashboard  
- Music Survey Analysis Dashboard  
- Fitness Performance Dashboard  
- Spotify Analytics Dashboard  
- General Exploratory Data Dashboard  

## Key Features
- Interactive dashboards with filters and slicers  
- KPI metrics and visual analytics  
- Data cleaning and transformation in Power Query  
- Insight generation from real-world datasets  

## Tools Used
- Power BI  
- Excel  
- Kaggle datasets  

## Purpose
To practice data analysis, visualization, and storytelling using real datasets and Power BI.
